Portfolio Website – AbdelKhalek El Ouali
Over dit project

Mijn persoonlijke React portfolio met projecten, vaardigheden en achtergrond. Gemaakt met React en Vite.

Functionaliteiten

Homepagina met introductie

Projectenpagina met overzicht van projecten

Project Detailpagina met screenshots en beschrijvingen

Over mij-pagina met vaardigheden

Navigatie via menu

Responsief ontwerp voor desktop en mobiel

Technologieën

React | Vite | CSS / Tailwind | JavaScript (ES6+) | Git/GitHub

Projectstructuur
src/
 ├ components/   # Herbruikbare componenten
 ├ pages/        # Home, Projects, ProjectDetails, OverMij
 ├ data/         # Projecten en skills
 └ App.jsx       # Hoofdcomponent + routing

Installatie
git clone https://github.com/Abdelkhaleke01/Portfolio_Website.git
cd Portfolio_Website
npm install
npm run dev


Open in browser: http://localhost:5173
